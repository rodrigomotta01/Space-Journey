<h2>Running</h2>

<h3>First Method</h3>

- Download the folders: Code, Audio and Images

```pip install pygame-ce```

```python maingithub.py```

<h3>Second Method</h3>

- Download the folder Space Journey
- Run Space Journey.exe

<h2>Controls</h2>

- Arrow Up: Move up
- Arrow Down: Move down
- Arrow Left: Move left
- Arrow Right: Move right
- Space: Shoot

<h2>Dependencies</h2>

- Python 3
- Pygame 2.5.2
